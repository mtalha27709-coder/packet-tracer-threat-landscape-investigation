# Packet Tracer - Investigate a Threat Landscape



## 📌 Overview



This Cisco Packet Tracer project explores different types of cybersecurity threats and vulnerabilities that exist in modern network environments. The lab demonstrates how attackers exploit weak configurations, phishing attacks, open wireless networks, and DNS manipulation techniques to compromise systems and users.



The project focuses on practical investigation and analysis of security weaknesses while also understanding mitigation strategies used in real-world cybersecurity operations.



---



# 🎯 Objectives



## Part 1: Investigate a Network Configuration Vulnerability



* Analyze insecure guest wireless network configurations
* Investigate unauthorized internal network access
* Identify risks caused by weak/default router credentials
* Explore network segmentation and guest isolation issues
* Recommend security hardening solutions



## Part 2: Investigate a Phishing Malware Vulnerability



* Simulate a phishing campaign
* Understand social engineering techniques
* Analyze malware delivery through phishing emails
* Investigate malicious website redirection
* Learn organizational phishing defense strategies



## Part 3: Investigate Wireless Network & DNS Vulnerabilities



* Explore rogue access point attacks
* Understand DNS hijacking attacks
* Investigate man-in-the-middle attack techniques
* Analyze malicious DHCP/DNS configurations
* Learn wireless security best practices
